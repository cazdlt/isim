Welcome to PyISIM documentation!
=======================================================
PyISIM is a Python client for IBM Security Identity Manager (ISIM/ITIM) web services.

Tested on ISIM 7.0.1 FP13 and ISIM 7.0.2 FP2 

Due to API limitations some functionalities are served through ISIM's REST API and some other through ISIM SOAP Web Services.

Link to our `Github project <https://github.com/cazdlt/pyisim>`__
